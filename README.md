## Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Ruby 3.0.1 or newer
- Rails 7.0.4 or newer
- Git
- NPM
- PostgresSQL
- Web browser

### Setup
```
git clone git@github.com:KaskMIL/school_system_backend.git
cd school_system_backend
```

### Install

Run `bundle install`

### Usage

Execute the following commands:
- rails db:create
- rails db:migrate
- rails db:seed
- rails s
